**Root Causes:**

- Neither prompt emphasizes strongly enough that simulation/iteration = Single Graph

- JSON output format needs stronger enforcement (no code blocks, ever)

**Planned Improvements:**

- Strengthen JSON format enforcement with negative examples
- Add clarification that algorithm problems with constraints are always Single Graph

For v2:

- Add explicit "NOT Divergent" examples section
